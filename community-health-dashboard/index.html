GitHub Token Setup

No GitHub token detected. While the dashboard works without authentication, you may hit rate limits (60 requests/hour). A token increases this to 5,000 requests/hour.

Setup Steps:

  1. Go to GitHub Settings ‚Üí Personal Access Tokens
  2. Click "Generate new token (classic)"
  3. Give it a name like "Community Health Dashboard"
  4. Select scope: public_repo (for public repositories)
  5. Click "Generate token" and copy the token
  6. Paste it below and click "Save"

Note: Your token is stored locally in your browser and never sent to any server.
